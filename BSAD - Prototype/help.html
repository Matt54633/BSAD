Help

Welcome to the help page. Find below tips on how to use the RMS.

  1. Navigate the system using the navbar and the buttons on each page
  2. When a material change occurs. A notification dot will appear on the Manage Alerts button on the home page
  3. To generate a report. Click on the 'Generate Reports' button. This will take you to Excel.
  4. In the Application Management page, you can edit apps, create them and assign RTOs and Traps to applications.
